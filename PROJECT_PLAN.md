# Job Tracker MERN

## Goal

A full-stack web application that helps users track job applications.

## Features

### Authentication
- Register
- Login
- Logout

### Jobs
- Create Job
- View Jobs
- Edit Job
- Delete Job

### Search & Filter
- Search by company
- Filter by status

### Dashboard
- Total Jobs
- Applied
- Interview
- Rejected
- Offer

### Profile
- Update profile
- Upload profile image

## Tech Stack

Frontend:
- React
- Tailwind CSS
- React Router
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Authentication:
- JWT
